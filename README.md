# 2048 웹게임

간단하고 재미있는 2048 퍼즐 게임입니다.

## 게임 방법

### 데스크톱
- **화살표 키**를 사용하여 타일을 이동하세요
- **Ctrl+Z** (또는 Cmd+Z)로 뒤로 무르기

### 모바일
- **스와이프 제스처**로 타일을 이동하세요
  - 위로 스와이프: 위로 이동
  - 아래로 스와이프: 아래로 이동
  - 왼쪽으로 스와이프: 왼쪽으로 이동
  - 오른쪽으로 스와이프: 오른쪽으로 이동
- **뒤로 무르기 버튼**으로 이전 상태로 되돌리기

### 공통
- 같은 숫자의 타일이 만나면 합쳐집니다
- **2048**을 만들면 승리합니다!
- 더 이상 움직일 수 없으면 게임 오버입니다
- 게임 오버 후에도 뒤로 무르기로 계속 플레이 가능!

## 기능

- ✅ 반응형 디자인 (모바일/데스크톱 지원)

## 실행 방법

### 로컬에서 실행

#### 방법 1: 간단한 HTTP 서버 (권장)
Python이 설치되어 있다면:

```bash
# Python 3
python -m http.server 8000

# 또는 Python 2
python -m SimpleHTTPServer 8000
```

그 다음 브라우저에서 `http://localhost:8000` 접속

#### 방법 2: Node.js 사용
```bash
# http-server 설치 (한 번만)
npm install -g http-server

# 서버 실행
http-server -p 8000
```

#### 방법 3: VS Code Live Server
VS Code를 사용하는 경우:
1. Live Server 확장 설치
2. `index.html` 우클릭 → "Open with Live Server"

#### 방법 4: 직접 열기 (제한적)
- `index.html` 파일을 브라우저에서 직접 열기
- ⚠️ 일부 기능이 제한될 수 있습니다 (CORS 정책)
